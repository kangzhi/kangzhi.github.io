Project-AR-Implementation

发表于 2019-02-15 | 分类于 Project

美国某公司的桌面式沉浸系统有一个很有趣而且很具实用价值的工具，基于自家硬件做的。虽然本身桌面式的沉浸设备本来就是扯淡，肯定是要被淘汰的，但是抛开局限于桌面有限空间这个点，不得不说它的思想做的还是比较有意思，在局部实现了真正意义上的虚拟世界和显示世界融合。
官方的视频效果图：
在这片文章中，利用opencv和opengl实现了技术上的整个流程，最后介绍了结合unity实现无缝将其他引擎内场景内容渲染过来的步骤和方法。整个项目耗时半年，对引擎底层的渲染技术、数据通信和交互、windows平台的编程机制以及cpp都有很好的理解和实现。由于本身接触这几块的内容不多，期间搜索了大量的资料，过去也一年多了，所以尽可能从实现思路将整个流程简要叙述。
